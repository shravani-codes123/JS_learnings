# 🎮 Number Guessing Game

## 📌 Description
This is a simple console-based number guessing game built using JavaScript.  
The system generates a random number, and the user tries to guess it within limited attempts.

## 🚀 Features
- Random number generation
- Limited attempts
- Input validation
- Feedback (Too high / Too low / Correct)
- Game over condition

## 🧠 Concepts Used
- Variables
- Functions
- Conditional statements
- Math.random()

## ▶️ How to Run
1. Open terminal
2. Run:
   node number-guessing-game.js

## 🧪 Example Output
Too low! Try again.
Attempts left: 4

Too high! Try again.
Attempts left: 3
